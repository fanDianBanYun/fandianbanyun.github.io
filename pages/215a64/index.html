5月27日新闻

# 中国新闻：

# 香港警务处处长促香港制止“软对抗”：

他说，虽然当前香港社会平静，但仍有暗涌，有人仍在尝试以软对抗的方式，煽动他人抗争。他受访时说，基本法赋予了香港居民享有集会、游行的自由。但香港警方在审批游行集会申请时，必须考虑对国家安全、公众安全、他人自由权利的影响。如若发现违法行为，必定果断执法。他指出，在 2019 年香港反修例风波发生后，很多游行集会都被 “骑劫”。警方绝不容许同样情况再次发生，因此警方审批每宗申请时，都会仔细分析情报，以免有任何人企图 “骑劫” 游行集会，危害到国家及社会安全。

翻评

这个我们熟，社会对抗不存在，就开始要把对抗扼杀在萌芽状态，这样禁止正常的游行活动，以及各种审查就开始变得合理了。只不过香港现在要完全审查网络和每一个居民的手机，达到我们的状态，可能还需要很长的时间。

# 英国外相促中国废除港版国安法：

英国外相祁湛明向国会提交新一份《香港半年报告书》，指北京仍然不遵守《中英联合声明》，呼吁北京和香港废除香港国安法。对此，香港政府强烈反驳，敦促英国停止干涉“纯属中国内政”的香港事务。

# 特斯拉车主维权名誉侵权案败诉：

历时两年的特斯拉车顶维权车主起诉特斯拉侵犯名誉权案、特斯拉侵犯个人隐私案终于落槌，一审法院均驳回了涉事车主张女士的诉讼请求，特斯拉胜诉。相关民事判决书显示，针对张女士起诉特斯拉侵犯名誉权一案，法院认为，张女士购买的特斯拉汽车使用一年后发生交通事故，她对产品质量提出质疑，因产品质量争议协商未果，张女士将喷涂 “刹车失灵” 字样的事故车辆停放至公共场所，之后又在上海车展身穿印有 “刹车失灵” 字样的上衣登上展车车顶扰乱公共秩序。

# 中国特使李辉周五抵达莫斯科访问：

与俄外长拉夫罗夫会谈。俄外交部称，二人就解决乌克兰冲突的前景举行了磋商。此前，李辉在欧洲之行框架下已访问乌克兰、波兰、法国、德国等。北京委任李辉为特使，寻求政治解决乌克兰冲突问题。李辉曾任中国驻莫斯科大使。自俄罗斯入侵乌克兰以来，中俄两国的外交与经济关系加强。北京也拒绝对俄罗斯作出谴责。乌克兰外长上周在与李辉会晤后表示，基辅不会接受"失去领土"的解决方案。

# 中国特使李辉促欧洲接受俄罗斯占领乌克兰领土：

据称，中国政府的特别代表李辉试图说服欧洲对话者，他们应该呼吁在乌克兰实行停火，并承认被占领的乌克兰领土为俄罗斯所有。华盛顿邮报综合欧洲相关官员说："中国大使被派遣推动北京对乌克兰的和平计划，传达了一个明确的信息：欧洲方面应该摒弃美国，坚定自身的自治权并立即呼吁停火，让俄罗斯保留其现在占领的邻国的部分领土，这是西方官员在整个李辉的行程中熟悉的论调。”

# 中国人在台不享有司法国民待遇：

有中国男子在台遭路灯电死，此案获台湾法院判定国赔，引发争议。台湾行政院週四通令，“大陆地区人民亦为中华民国人民”的函释已过时，应即日起停用。台湾行政院长陈建仁称，这是为了避免国际误解。台湾行政院週四宣布，所谓“大陆地区人民亦为中华民国人民”的相关函释，应即日起停止适用。这意味著，台湾官方认定：中国大陆人民不具中华民国国籍、并非国民，也因此不享有或负担中华民国国民的权利义务。

# 河北发生烟花爆竹爆炸事故：

河北省廊坊市大城县臧屯派出所民警在对涉嫌非法买卖的烟花爆竹进行取样鉴定时发生爆炸。经初步核查，现场涉及人员11人。目前，已搜救出9人，其中5人正在救治，4人已无生命体征；失联2人，正在全力搜救。善后处置工作正在推进。

# 中国拒绝美国设立热线机制：

中国和美国似乎再次陷入对立。中国间谍气球的争议仍在持续，美国现在指责北京拒绝或未回应其建立热线沟通的请求。国防部印度 - 太平洋安全事务助理秘书表示，美国非常愿意与中国保持开放的沟通渠道，以避免在和平和危机期间出现误解。

# 阿迪达斯寻求重振中国市场：

德国运动品牌阿迪达斯两年前因新疆棉事件被中国消费者抵制后，阿迪达斯正尝试修复公司在中国的形象，以扭转中国区业绩连续七个季度下滑的局面。自新疆棉争议爆发后，阿迪达斯开始赞助一些有中国政府背景的活动，同时为官方文体项目提供支持，包括协助制作一档颂扬中国体育英雄的电视节目。该公司还在与中国体育明星合作，开发更多专门针对当地消费者的产品，寻求改善与中国及其消费者的关系。

# 亚洲新闻：

# 日本长野县中野市凶杀加害者因猜忌行凶：

致命凶案嫌犯星期五供称，因怀疑 “被说坏话” 和 “会被杀”，所以持枪支和利刃杀害四人。共同社报道，凶嫌是中野市议会议长 31 岁从事农业的儿子青木政宪，他在先后杀害两名警员和两名老妇后，躲进住家与警方对峙了 12 小时。青木在被捕后供称：“我认为受害女性说我的坏话，因此杀了她。我觉得自己会被枪杀，所以也杀了警察。”

# 中国谴责日本参加北约峰会：

中国对日本首相预计在 7 月份参加北约峰会表示谴责，称东京不应做任何破坏地区国家之间相互信任的事情。外交部发言人毛宁在北京的一次新闻发布会上表示，北约向亚太地区的东部进军 "破坏了地区的和平与稳定"，该地区的国家 "应该保持高度警惕"。毛宁敦促日本 "汲取历史教训，致力于和平发展"。

# 中越再次南海对峙：

越南外交部要求中国从东南亚国家海域撤走一艘调查船以及海警船和渔船。中国外交部星期五回应说，中国有关船只在本国管辖海域开展正常的活动正当合法。在南中国海问题上，中国同有关各方都保持着沟通，同时也将坚定地维护中方的合法权益。中国外交部发言人毛宁星期五回答相关提问时说，中国对南沙群岛及其附近海域拥有主权，对相关海域拥有主权权利和管辖权。中国的有关船只在本国管辖海域开展正常的活动正当合法，不存在进入他国海域或 “专属经济区” 的问题。

翻评

本来在我国与西方交恶之时，对东盟外交关系应该是一个关键方向，但与菲律宾、越南、马来西亚等过的南海主权争议，会成为我们与东盟关系的关键威胁。这已经导致菲律宾大面积倾向西方，下一步就看越南了，越南与我们有复杂的历史关系。

# 泰国旅游局计划把前进党党魁营造为泰国的 “旅游大使”：

希望在全球掀起一股 “披塔热潮”，以争取更多国际游客到泰国旅游。42 岁的披塔正在寻求成为泰国的下一任首相。泰国《民族报》报道，泰国旅游局局长育塔萨说，披塔是担任泰国旅游大使的最佳人选，因为他深受年轻人的欢迎。育塔萨说，披塔可以协助泰国吸引更多外国游客前来泰国，尤其是中国、韩国和日本的年轻游客。

# 泰国寻求与中国共同投资东盟能源项目：

关注着邻国动态的泰国将目标对准与中国大湾区的高科技合作。泰国正在寻求中国共同投资在老挝和越南等国的风力涡轮机、水电大坝和太阳能农场。泰国也是发展跨亚洲高速铁路网络的关键合作伙伴，该网络将从昆明延伸到新加坡。

翻评

中国在东盟建构铁路网络，甚至源起与一带一路倡议之前，在2006年就与各国建立了“泛亚铁路网协议”的备忘录，后成为一带一路的关键项目，其中与老老挝的从昆明连接万象的铁路已经在2021年通车，其他的尚在建设中，也不断引发争议。

# 韩亚航空一飞机在空中被乘客打开舱门：

据韩国航空业界消息，韩亚航空公司一架由济州飞往大邱的客机在降落前发生舱门打开事故。韩联社报道，韩亚航空OZ8124航班星期五上午 11 时 49 分从济州机场起飞，在降落大邱机场前的 12 时 45 分舱门突然打开。据悉，客机在舱门打开状态下降落跑道。韩亚航空发言人过后发表声明说，客机在大邱机场安全降落。这架机上共载有 194 名乘客，所幸无人受伤，但部分乘客受到惊吓后出现呼吸困难等症状，降落后立即被送医，无人丧失意识。韩亚航空相关人士透露，坐在紧急出口座位的乘客供述时称触碰到舱门把手，警方正就这起事件展开调查。

# 科技新闻：

# 张文宏团队获科普特等奖：

2022 年度上海市科学技术奖获奖名单星期五上午正式公布，共计授奖 316 项，其中复旦大学附属华山医院感染科主任张文宏带领的团队获得 “科学技术普及奖” 类别中唯一的 “特等奖”。

# 百度将推出文心3.5：

百度 CEO 李彦宏说，百度很快会正式推出文心大模型的 3.5 版本。综合网媒 “DoNews” 和新浪科技报道，李彦宏星期五在中关村论坛发表题为《大模型改变世界》演讲。李彦宏说，当前正处在一个以大模型为核心的人工智能新时代，大模型改变了人工智能，大模型即将改变世界。

翻评

从最初闹笑话后，就一直没有再多关注文心了，有经常使用的人吗？现在到什么水平了？

# Neuralink获得人体临床资格：

马斯克的Neuralink公司25日宣布，它已经获得美国食品和药物管理局的批准，可以对人体进行首次临床试验。公司并未透露研究的目的，仅表示尚未开始招募参与者，但将很快提供相关计划的信息。据Neuralink的员工称，该公司在2022年初曾寻求FDA的批准，但因安全原因被拒绝。

# 印度TATA进军AI领域：

印度的塔塔公司寻求借助人工智能帮助亚洲应对老年护理、气候变化和技能培训。IT 服务提供商塔塔咨询服务公司计划在新加坡、马来西亚、泰国等国家使用人工智能来为公众提供更流畅的服务。对于许多偏远地区，可以访问多语言人工智能可能有助于帮助老年人和 / 或残疾人，同时也能记录对应对全球变暖有用的数据。

翻评

这个塔塔是印度的一家超大集团，印度第一财团，其产值相当于印度GDP的6%，涉足钢铁、航空、科技、汽车、纺织、通信等等的诸多领域，最近还宣布进入芯片领域，开始印度芯片国造计划。

# 中国再出手工打磨芯片技术：

中国近期公布的由 Powerstar 制造的 x86 国产 CPU 不过是重新标签的英特尔 10 代芯片。中国即将推出的 x86 国产 CPU规格与Intel的Comet Lake Core i3-10105相同。中国的一家芯片制造商 PowerLeader，近期发布了其全新的代号为 PowerStar 的 x86 CPU。该公司举行了一次大型活动，邀请了来自全国各地的媒体参加，结果发现，这个芯片不过是重新标签的英特尔第 10 代 CPU。

翻评

手工打磨CPU已经成为中国芯片业的传统艺能了，但这个骗局竟然可以再次发生，令人惊讶。

# 财经新闻：

# 大量劳动力涌入网约车：

中国 4 月新增超过 17 万名网约车司机，创下放开疫情管控以来的最高环比涨幅，但订单数量却下滑，导致市场供过于求。深圳等多地交通运输部门接连发布网约车饱和预警，有的甚至暂停受理网约车经营许可。据中国网约车监管信息交互系统最新统计，中国在 4 月份新增 17 万 7000 名网约车持证司机，环比增长 3.4%，使全国数量达到了 540 万 6000 人。单在今年首四个月，中国新增了超过 31 万名网约车司机。不过，订单数量在 4 月环比下跌 1000 万，意味着网约车市场在当月出现司机多、乘客少的局面。

翻评

灵活就业真的对整体经济有好处吗？在削减失业率方面也许确实如此，但大量低成本的Bullshti Job对劳动力和劳动者具有一种“惰性”的吸引力，其实压抑了其通过其他方式改善生活的可能。这个Easy Choice当然是一种社会兜底的选择，但对整体经济活力可能是有很大影响的。

# 太阳能成为首选新能源：

分析师表示，随着中国的能力不断扩大，太阳能将在国内外成为可再生能源的主导力量。香港分析师表示，预计中国的太阳能行业今年将新增 120GW 到 140GW 的容量，至少比 2022 年高出 40%。在上海的一个活动上听闻，太阳能发电将很快成为中国最经济的电力形式，超过水电成为今年底中国最大的非化石能源。

翻评

这个选择的背景尤其与气候危机有关，全球干旱导致的水资源短缺，与气候系统紊乱导致的南北风力交换减少，让水电和风电战略遭遇重挫，因此转向太阳能也是如此考虑。但光伏电池板制造过程的高污染，太阳能发电需要大规模土地面积，发电本身的间歇和随机等，也是太阳能发电大规模铺开的障碍。
